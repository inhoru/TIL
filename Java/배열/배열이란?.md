 # 1. 배열
 - 배열은 같은 데이터타입의 여러 변수를 하나의 묶음으로 만드는것을 말한다.


   - 예를 들어 변수 100개가 각각 존재한다면 이 100개의 변수는 모두관련이없는 각각고유한 변수이다.
   - 그런데 그의 반해 **배열은 처리하면 연속적인 변수로 처리**가 된다. 
   - 그래서 효율적으로 프로그래밍이 가능하다.

- 그러므로 **배열**로 만든다면 **모든변수가 연속적으로 이어져 있는 것을 알 수있다.**
