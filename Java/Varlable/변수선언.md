# 1. 변수를 왜 선언할까?


-#### 변수를 사용하지 않을때

```java
System.out.println(2*11234564*45464564);
System.out.println(2*75564464*754564564);
System.out.println(2*9665454*415615644);
```


-#### 변수를 사용하지 할때

```java
System.out.println(2 * p * r);
System.out.println(2 / r * 7);
System.out.println(2* r * d);
```
