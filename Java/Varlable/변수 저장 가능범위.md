# 1. 데이터 저장 단위

  - #### 저장 공간이 제한적이기 때문에 저장 크기에 대한 기준과 CPU가 데이터를 처리할 때 일정한 기준 필요하다.<br/><br/>


	## 비트(bit)

	- 컴퓨터가 나타내는 데이터의 저장 최소 단위다.
	- 2진수 값 하나를 저장할 수 있는 메모리공간을 의미

	## 바이트(byte)
	
	- 데이터 처리 또는 문자의 최소 단위다.
	- 8개의 비트가 모여 하나의 바이트가 구성된다.
